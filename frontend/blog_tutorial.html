AISTONE平台使用教程

AISTONE 2025年9月 约8分钟阅读

🚀 快速开始

步骤1：访问平台

打开浏览器访问 https://aistone.org，无需注册即可开始使用。

步骤2：选择生成类型

在页面上选择您想要生成的内容类型：

  • 生成图片：将文本描述转换为高质量图片
  • 生成语音：将文本转换为自然流畅的语音

步骤3：输入描述文本

在文本输入框中输入您的描述：

  • 支持中文和英文
  • 描述越详细，生成效果越好
  • 可以使用示例按钮快速填充

步骤4：设置参数

图片生成参数：

  • AI模型：FLUX（高质量）、Turbo（快速）、Kontext（图像编辑）
  • 宽高比例：正方形、横向、竖向等多种选择
  • 生成数量：1-4张图片
  • 去除水印：获得更清洁的图片

语音生成参数：

  • 音色选择：6种不同的AI音色
  • 语速控制：0.25x到4.0x调节

步骤5：点击生成

点击"开始生成"按钮，AI会处理您的请求。通常需要10-30秒。

🎨 图片生成功能详解

AI模型选择

FLUX - 高质量艺术创作

  • 最新的开源模型
  • 优秀的细节表现
  • 适合艺术创作和专业用途

Turbo - 快速生成

  • 生成速度最快
  • 适合快速原型和测试
  • 质量略低但效率高

Kontext - 图像到图像

  • 支持图像编辑功能
  • 可以基于现有图像生成
  • 适合图像变换和修改

宽高比例设置

  • 正方形 (1:1)：适合社交媒体头像、LOGO
  • 横向 (16:9)：适合横幅、背景图
  • 竖向 (9:16)：适合手机壁纸、海报
  • 标准 (4:3)：经典比例，适合大多数用途

质量优化技巧

  1. 使用详细描述：描述越详细，生成效果越精确
  2. 选择FLUX模型：获得最佳质量
  3. 使用高分辨率：选择较大的图像尺寸
  4. 利用智能优化：使用平台的优化功能
  5. 添加负面提示词：排除不想要的元素

🎵 语音合成功能详解

音色选择指南

Nova - 清晰专业的女声

适合：商务演示、教育内容、正式场合

Echo - 温暖友好的男声

适合：播客、故事朗读、休闲内容

Fable - 富有表现力的叙事声音

适合：有声书、故事讲述、戏剧化内容

Onyx - 深沉权威的男声

适合：新闻播报、严肃内容、权威解说

Shimmer - 明亮活泼的女声

适合：儿童内容、轻松话题、活泼解说

Alloy - 平衡多用途的声音

适合：通用内容、多场景应用

语速控制技巧

  • 0.25x-0.5x：适合学习、重要信息强调
  • 0.75x-1.0x：正常语速，适合大部分内容
  • 1.25x-2.0x：快速播报、时间紧张场合
  • 2.0x-4.0x：超快速度，特殊需求

💡 高级使用技巧

智能优化功能

点击"优化"按钮，AI会自动：

  • 将中文描述翻译为英文
  • 优化提示词结构
  • 添加质量增强词汇
  • 改善语法和表达

批量生成策略

选择生成多张图片时：

  • 每张图片会有不同的变化
  • 可以从中选择最满意的结果
  • 节省重复生成的时间

账号功能（可选）

注册账号后可以：

  • 保存高清图片（每日3张）
  • 查看生成历史
  • 管理个人作品
  • 获得更好的服务体验

❓ 常见问题解决

生成失败怎么办？

  1. 检查网络连接
  2. 简化描述文本
  3. 切换不同的AI模型
  4. 稍后重试

生成质量不满意？

  1. 使用更详细的描述
  2. 选择FLUX模型
  3. 添加质量关键词
  4. 使用负面提示词
  5. 尝试智能优化功能

速度较慢？

  1. 选择Turbo模型（速度优先）
  2. 避免高峰时段
  3. 减少生成数量
  4. 降低图像分辨率
