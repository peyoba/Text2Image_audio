How the platform is assembled

Image pipeline

  • Prompt validation and routing on Cloudflare Workers with rate limiting.
  • Generation powered by Pollinations FLUX, Turbo, and Kontext models.
  • Optional upscaling stored temporarily in Cloudflare R2 before download.

Voice pipeline

  • Client-side text normalisation before streaming to OpenAI Audio TTS.
  • Audio buffers delivered as WAV; nothing is stored server-side.
  • Policies mirror OpenAI content rules and block unsafe prompts.
